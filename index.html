krug CaaS

Krug is a Compiler-as-a-Service programming language. The source code for the project can be found here.

Give it a go

There is a server (api.krug-lang.org:8080) hosting an instance of the krug CaaS. Feel free to give it a go at home: 

$ go get github.com/krug-lang/krugc
$ cd krugc
$ go build
$ ./krugc -server="api.krug-lang.org.com:8080" tests/hello_world.krug

Or optionally, you could clone the server and run it locally. The 'krugc' driver will run locally by default: 

$ go get github.com/krug-lang/krugc-api
$ cd krugc-api
$ go build
$ ./krugc-api
