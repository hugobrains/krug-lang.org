Krug is a programming language for systems programming. The compiler is a compiler-as-a-service, which means that the compilers runs as a server. Each compilation phase is cut up into an API route which can be invoked from a client.

Releases Installer

Features:

  • Fast
  • Statically Typed
  • Optional garbage collection
  • Type inference
  • Minimal runtime
  • Native Language Server
  • No nonsense
